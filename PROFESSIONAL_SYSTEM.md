## 🚀 **איך להפעיל**

### הפעלה מהירה
```bash
# Backend
cd backend
docker-compose up --build

# Frontend (בטרמינל נפרד)
cd frontend
npm install
npm run dev
```

### גישה לאפליקציה
- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs
